
# STM32F407 简易示波器与信号源
本项目以STM32F407最小系统板为核心，搭建了一套集简易示波器、信号源于一体的综合系统。下位机依托ADC模块完成模拟电压信号的采集，并通过串口将数据传输至PC端；上位机基于Python、PyQt5与PyQtGraph开发，可实现波形实时可视化展示，同时集成采集控制、信号源调控、波形类型切换、时基调节、电压量程设置及触发参数配置等核心功能。

项目最终演示以上位机软件按钮操作为核心控制方式，板载物理按键功能未纳入最终交付演示环节。

## 项目功能
1.  基于STM32F407的ADC模块，实现模拟输入电压信号的实时采集
2.  借助DAC模块，输出正弦波、三角波、方波三种标准波形
3.  通过串口通信协议，完成下位机与上位机之间的数据交互
4.  上位机实时绘制并展示电压-时间关系波形
5.  上位机支持采集功能的启动与停止控制
6.  上位机支持信号源输出的开启与关闭控制
7.  上位机可自由切换输出波形的类型
8.  上位机支持时基参数、电压显示范围、触发模式及触发电平的灵活调节



## 硬件连接
进行信号源与示波器闭环测试时，需按以下方式完成硬件接线：
PA4（DAC输出引脚） → PA5（ADC输入引脚）
系统地（GND） → 系统地（GND）

串口通信采用下位机工程中的USART1串口，通信波特率固定为：
115200

## 上位机使用方法
### 安装运行依赖
进入上位机软件目录，执行依赖安装命令：
```bash
cd "上位机软件"
python -m pip install -r requirements.txt
```

### 启动上位机程序
```bash
python oscilloscope.py
```

### 操作使用步骤
1.  在界面中选择对应的串口端口（如COM13）
2.  波特率参数选择115200
3.  点击【开始采集】按钮，启动信号采集功能
4.  点击【开启信号源】按钮，启动波形输出功能
5.  通过下拉菜单选择需要输出的波形：正弦波、三角波或方波
6.  调节时基（ms/div）和电压范围（V）参数，优化波形显示效果

## 下位机烧录方法
进入下位机工程根目录，执行烧录命令：
```bash
cd "下位机"
make flash
```

烧录完成的标准标识：OpenOCD控制台输出包含以下内容：
** Programming Finished **
** Verified OK **
** Resetting Target **

## 通信协议
### 上位机发送指令（单字节格式）
| 命令值 | 对应功能 |
| ---- | ---- |
| 0x01 | 启动信号采集 |
| 0x02 | 停止信号采集 |
| 0x10 | 开启正弦波信号源 |
| 0x11 | 开启三角波信号源 |
| 0x12 | 开启方波信号源 |
| 0x13 | 关闭信号源 |

### 下位机上传数据帧格式
`AA BB LEN_H LEN_L 数据段 CC DD`
其中数据段为12位ADC采样值，传输遵循高字节在前、低字节在后的规则；上位机通过换算公式`3.3V / 4095`，将ADC原始数值转换为实际电压值。

## 测试结果
经实际测试验证，上位机可正常完成串口打开、控制指令发送、下位机数据接收等操作。将PA4与PA5短接完成闭环测试后，上位机能够清晰显示DAC输出、ADC采集的完整波形。

实测现象：
1.  关闭信号源时，ADC输入端显示稳定直流电压或悬空状态的噪声信号
2.  开启正弦波信号源后，上位机界面呈现规则的周期性正弦波形
3.  开启三角波信号源后，上位机界面呈现规则的周期性三角波形
4.  开启方波信号源后，上位机界面呈现高低电平交替切换的方波波形

上位机界面实时同步显示数据包数、采样点数、发送指令数、接收字节数，以及最新电压范围参数。

## 已知限制
1.  本项目主要用于课程设计与实验教学，测量精度、采样带宽均有限，无法替代专业商用示波器
2.  项目最终演示仅支持上位机按钮控制，板载按键功能暂不做演示
3.  ADC输入电压需严格控制在STM32F407额定范围（0~3.3V）内，防止芯片硬件损坏
4.  进行信号源与示波器闭环测试时，需确保PA4与PA5引脚接线正确无误
